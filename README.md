# binary-script-encoder-decoder

usage:
    
    encoder:
	any type of text.
    
    decoder:
	the programme reads only binary, kindly refrain from using other inputs
	use <space> to separate the letters, and <slash>(/) to words(quotes are not needed)
	
	e.g : "1010" or "0110 0111" or "100010 01001/01101 01101"

	
	FOR THE FIRST USE TRY THIS in decoder :
	1110100 1101000 1100001 1101110 1101011 1111001 1101111 1110101/1100110 1101111 1110010/1110100 1110010 1111001 1101001 1101110 1100111 101110
